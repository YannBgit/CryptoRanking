# quick start

## server installation

```cd server```

```npm install --force```

```npm run dev```

## client installation 

```cd client```

```npm install --force```

## execution du script tailwind

```npm run build-css```

## execution de react

```npm start```

vérifier si le proxy communique bien, sinon vérifié si le port 5000 est bien sur écoute
